# STATE.md template

Write to `.planning/STATE.md`. A ~4-line cursor, overwritten in place, never
appended. No audit logs, no activity tables, no progress bars - git history
is the log.

<template>

```markdown
# State

Phase: [N] of [total] ([phase name])
Status: [ready to plan | planning | executing | verifying | phase complete]
Next: [the one command to run next, e.g. /cad-context 1]
```

</template>
